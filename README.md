# Install

```sh
git clone git@github.com:150years/maryana.design
cd maryana.design
git clone git@github.com:150years/maryana.design _site
cd _site
git checkout gh-pages
cd ..
```
# Develop

```sh
bundle exec jekyll serve
```

# Build

```sh
./build.sh
```
